HyperFurion VK

A voice keyboard for Linux. Press Ctrl+Space, speak, and it types into whatever app has focus.

$ recording

How it works

A local daemon captures microphone audio, sends it to the speech-to-text provider you configured, and types the transcript into the focused app through a uinput virtual keyboard. It works in terminals, IDEs, browsers — anything that takes keystrokes, on Wayland and X11.

It also does the reverse: select text anywhere and run voice-keyboard tts to hear it read aloud.

Usage

# tap to toggle dictation, or hold to talk
Ctrl+Space

# equivalent CLI
voice-keyboard toggle
voice-keyboard status

# read the selected text aloud
voice-keyboard tts

Providers

Bring your own API key. Switch providers with one line in ~/.config/voice-keyboard/config.toml.

  • Speech-to-text: xAI (streaming), OpenAI, Groq, Deepgram, AssemblyAI
  • Text-to-speech: xAI, OpenAI, ElevenLabs

Notes

  • Linux only. Requires Python 3.11+, systemd user services, and uinput access.
  • Cloud STT/TTS — this is not an offline dictation engine.
  • Recording overlay is GNOME Shell 50 on Wayland; other desktops get notifications.
  • Hotkeys need input-group access; log out and back in once after install.
